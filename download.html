Installation guide

Installing on Windows

1

Download and unzip

Download the zip file above and extract it. You should see a folder called com.captioncraft.panel.

2

Copy to the CEP extensions folder

Move the com.captioncraft.panel folder into:

C:\Users\YourName\AppData\Roaming\Adobe\CEP\extensions\

Can't see the AppData folder? In File Explorer, go to View → Show → Hidden items. AppData is a hidden folder.

If the extensions folder doesn't exist, create it.

3

Enable unsigned extensions

Adobe requires a registry edit to allow third-party CEP plugins. Open Registry Editor (search for regedit in the Start menu) and navigate to:

HKEY_CURRENT_USER\SOFTWARE\Adobe\CSXS.12

Create a new String Value named PlayerDebugMode and set its value to 1.

If CSXS.12 doesn't exist, try CSXS.11 or CSXS.13 — the number matches your AE version. You only need to do this once.

4

Open the panel in After Effects

Launch After Effects (or restart it if it was open). Go to:

Window → Extensions → Captio by Keyweaver

The panel will open on the right side. Sign in with your Keyweaver account and you're ready to generate.

Installation guide

Installing on macOS

1

Download and unzip

Download the zip file above. Double-click it to extract. You should see a folder called com.captioncraft.panel.

2

Copy to the CEP extensions folder

Move the folder into:

~/Library/Application Support/Adobe/CEP/extensions/

Can't see the Library folder? In Finder, hold Option and click the Go menu — Library appears in the list.

If the extensions folder doesn't exist, create it.

3

Enable unsigned extensions

Open Terminal and run:

defaults write com.adobe.CSXS.12 PlayerDebugMode 1

Replace CSXS.12 with your version if needed. You only need to do this once.

4

Open the panel in After Effects

Launch After Effects (or restart it). Go to:

Window → Extensions → Captio by Keyweaver

The panel will open. Sign in with your Keyweaver account and you're ready to generate.

Troubleshooting

Common questions

The panel doesn't appear in Window → Extensions
Make sure you've set PlayerDebugMode to 1 in the registry (Windows) or via Terminal (Mac), then fully quit and relaunch After Effects. Restarting AE is required — it doesn't hot-reload extensions. Also double-check the folder is named exactly com.captioncraft.panel with no extra nesting.
I see a blank white panel
This usually means the CEP debug flag isn't set. Re-run the registry edit (Windows) or Terminal command (Mac) and restart AE. If the panel still shows blank, right-click inside it and choose Inspect — check the console for errors.
After Effects says the extension is not compatible
Captio requires After Effects 2021 or later. If you're on an older version, you'll need to update AE. You can check your version under Help → About After Effects.
Transcription fails or returns an error
Check that you're signed in and have Keyweaver Credits available. Files must be under 25 MB on the free and credits plans. If the file is larger, use the Export Audio option in AE to create a smaller audio-only export first, then generate captions from that.
Can I use Captio on multiple computers?
Yes. Install the plugin on as many machines as you like. Your credits and subscription are tied to your Keyweaver account, not to a specific device. Sign in on each machine and your balance follows you.
