Installation guide

Installing on Windows

1

Download and extract

Download Cuemark-Install-win-v1.0.0.zip, then right-click it and choose Extract All. Open the extracted folder.

2

Run Install-Cuemark.cmd

Double-click Install-Cuemark.cmd. It copies Cuemark into your Adobe extensions folder and configures After Effects — no Registry Editor needed. A command window will open briefly while it installs.

3

Restart After Effects

Fully quit After Effects if it was open, then launch it again. Extensions only load on startup.

4

Open Cuemark

Go to:

Window → Extensions → Cuemark by Keyweaver

Sign in with your Keyweaver account and you're ready to generate.

Manual install (advanced)

Copy com.captioncraft.panel to %AppData%\Adobe\CEP\extensions\ and set PlayerDebugMode to 1 under HKCU\Software\Adobe\CSXS.12. The installer does this automatically.

Installation guide

Installing on macOS

1

Run the installer

Download the zip, extract it, and double-click Install-Cuemark.command. It copies Cuemark into your CEP extensions folder and enables third-party panels in After Effects — no Terminal commands needed.

If macOS says the installer is from an unidentified developer, right-click Install-Cuemark.command → Open once.

2

Restart After Effects

Fully quit After Effects if it was open, then launch it again.

3

Open Cuemark

Go to:

Window → Extensions → Cuemark by Keyweaver

Sign in with your Keyweaver account and you're ready to generate.

Troubleshooting

Common questions

Windows blocked the installer or showed a security warning
Extract the zip fully, then run Install-Cuemark.cmd from the extracted folder (not from inside the zip). If Windows warns about the script, click More info → Run anyway.
The panel doesn't appear in Window → Extensions
Run the Cuemark installer again, then fully quit and relaunch After Effects. Restarting AE is required — it doesn't hot-reload extensions. If you installed manually, check the folder is named exactly com.captioncraft.panel with no extra nesting.
I see a blank white panel
Re-run the Cuemark installer and restart AE. If the panel still shows blank, right-click inside it and choose Inspect — check the console for errors.
After Effects says the extension is not compatible
Cuemark requires After Effects 2022 or later (manifest host version 22.0+). If you're on an older version, you'll need to update AE. You can check your version under Help → About After Effects.
Transcription fails or returns an error
Check that you're signed in and have Keyweaver Credits available. Each generate is limited to 15 minutes of footage. Audio uploads are capped at 4 MB; Cuemark extracts audio from video automatically. If extraction fails, export audio-only (MP3/WAV under 4 MB) from your footage and generate again.
Can I use Cuemark on multiple computers?
Yes. Install the plugin on as many machines as you like. Your credits and subscription are tied to your Keyweaver account, not to a specific device. Sign in on each machine and your balance follows you.
macOS blocked Install-Cuemark.command
Extract the zip fully first. Then right-click Install-Cuemark.command → Open (not double-click the first time). macOS may also quarantine files downloaded from the web — the installer clears quarantine when it runs. If it still will not open, run xattr -cr . in the extracted folder, then try again.
