Using Cuemark

Everything you need to go from footage to finished captions inside After Effects.

Watch the full walkthrough: install, generate, style, and export subtitles in After Effects.

Installation

Cuemark is a CEP panel for Adobe After Effects. The panel appears under Window → Extensions after you install it and fully restart AE.

Full step-by-step instructions are on the download page.

Windows (recommended)

Download Keyweaver Installer from the download page and run Keyweaver-Setup.exe. It installs Keyweaver Manager on your PC (signed by Keyweaver Ltd). Open Manager from the Start Menu, click Install on Cuemark, then fully quit and reopen After Effects.

If Windows shows SmartScreen, click More info → Run anyway and confirm the publisher is Keyweaver Ltd. No admin password is required.

Windows (zip fallback)

Prefer not to run an .exe? Download the zip from the download page, extract it, and double-click Install-Cuemark.cmd. Click Run if Windows says the publisher could not be verified. The installer copies files to your user AppData folder only.

Manual fallback: copy com.captioncraft.panel to %AppData%\Adobe\CEP\extensions\ and double-click Enable-Cuemark-CEP.reg once (both included in the zip).

macOS

Extract the zip and double-click Install-Cuemark.command. If macOS blocks it, right-click → Open → Open once.

Manual fallback: copy the panel folder to ~/Library/Application Support/Adobe/CEP/extensions/ and run defaults write com.adobe.CSXS.12 PlayerDebugMode 1 in Terminal (see README.txt).

The Mac installer includes ffmpeg for video transcription - no Homebrew or extra setup needed. If video generate still fails, re-download from the download page and run the installer again.

After installing, you must fully quit and relaunch After Effects. Restarting is required - closing and reopening a panel is not enough.

Your first generate

1

Open the panel

In After Effects, go to Window → Extensions → Keyweaver - Cuemark. Sign in with your Keyweaver account.

2

Open your edit comp

Open the composition you want subtitles in (your main edit comp with video or audio - not the Cuemark Subtitles precomp). If several layers have footage, you will choose which layer to transcribe when you generate.

3

Click Generate Subtitles

On the Generate tab, click Generate Subtitles. A confirmation shows duration and credit cost (1 credit per started minute). Cuemark transcribes the audio, then builds native text layers in your comp.

4

Edit and style

The panel switches to the Text tab. Review and save cue edits, then use Presets and Style to change look and animation. Style changes apply to your AE layers automatically.

Captions are placed in a precomp called Cuemark Subtitles inside your edit comp. One undo removes everything cleanly. Older projects may still show Captio Subtitles until you generate again - Cuemark renames these automatically.

Credits & account

Your credit balance is shown in the panel header. 1 Keyweaver Credit = 1 minute of audio. Fractional minutes round up - a 5 min 20 sec file costs 6 credits.

Credits are only consumed when you generate captions. Re-applying a style, editing text, or switching presets costs nothing.

Top up or manage your subscription at keyweaver.io/account. The same balance works in the panel and on the website - sign in with the same email.

Open Cuemark in After Effects to try guest credits without an account. Sign up free for 40 Keyweaver Credits shared across Cuemark, Superconductor, Trillian, and Tamborine. After that, buy a credit pack or subscribe.

Generate tab

The Generate tab is where you kick off transcription.

Choosing source footage

With your edit comp active, click Generate Subtitles. Cuemark scans the comp for video and audio layers with files on disk. If there is one layer, it is selected automatically; if there are several, a picker asks which layer to transcribe. Duration and credit cost are shown in the confirmation before transcription starts.

Max words per line

Use the dropdown to control line length. The default is Auto (automatic line breaks). Pick a number to force shorter or longer lines. Change this and generate again to re-wrap the same transcript without spending credits.

Export subtitles (SRT / VTT / CSV)

After subtitles exist, the Generate tab shows Export subtitles with SRT, VTT, and CSV buttons. Exports use the text from your saved cues in the Text tab.

Re-generating

If you generate again on the same footage, Cuemark uses the cached transcript - no credits are deducted for a second generate. Credits are only consumed when the audio is transcribed for the first time.

Clips are limited to 15 minutes per generate. Split longer footage into sections. Audio uploads are capped at 4 MB. Cuemark extracts audio from video automatically on both Windows and macOS (ffmpeg is bundled in the installer).

Text tab

The Text tab shows every caption cue as an editable text box. Click a cue to scrub the After Effects playhead to that moment. Edits are not written to AE until you save each line.

Saving edits

Each row has a Save button that turns blue when you've made a change. Click it (or press Ctrl+Enter) to write the edit to the AE layer. You'll see a spinner while the layer updates.

Tips

  • Transcription is highly accurate but occasionally mishears proper nouns, accents, or technical terms - the Text tab is the fastest way to fix these
  • Editing text does not consume credits
  • If you change the text case setting in the Style tab, it re-processes the original transcript - so editing in the Text tab first and then changing case is the correct order

Style tab

The Style tab controls the appearance of all caption layers. Most changes apply automatically after a short delay - there is no separate Apply button.

Typography

Use the font picker to browse installed fonts - each option renders in its own typeface. Font size, bold, italic, and text case (aa as generated, Aa title case, AA uppercase) are all controllable.

Colour

Set the text colour using the colour picker. For the background, toggle it on and pick a colour and opacity separately.

Background shape

Three shapes are available:

  • Rounded - rounded rectangle that hugs the text (pill shape)
  • Bar - full comp-width bar behind the text
  • Lower third - a solid block across the lower portion of the frame

Outline and shadow

Text outline adds a stroke around each character. Shadow adds a soft drop shadow. Both can be toggled independently and coloured separately from the text.

Position

Choose bottom, middle, or top. Use the vertical margin slider to fine-tune placement from the edge of the frame.

Number of lines

Switch between 1, 2, or 3 lines per cue. The character-per-line count adjusts automatically - tighter wrapping at higher line counts.

Presets tab

Presets are complete style configurations - font, colour, animation, background - applied with one click. Ten built-in presets ship with Cuemark:

  • Colour Highlight, Impact Words, Highlight Words, Word Banner, Box Cascade
  • Word Stamp, Typing Words, Drop Words, Elegant Highlight, Bold Fade On

Applying a preset rebuilds all AE layers with the new style. This does not consume credits - only transcription does.

Animations

Fourteen animation types are available from the Style tab dropdown (including None). All are applied as native After Effects text animators - fully editable after generation.

Animation What it does
NoneStatic text, appears instantly at cue in-point
RollWords rotate in on the Y axis, word by word
SpotlightEach word highlights in your accent colour as it's spoken, using word timestamps
HighlighterSpoken words get a coloured highlight bar behind them
ImpactEach word appears as a separate full-screen layer timed to speech
KaraokeProgressive word reveal synced to speech
Fade OnEach cue fades in softly over the cue duration
DropCharacters drop in from above
CascadeCharacters reveal left to right with a blur
Drift DownCue drifts down gently as it fades in
RiseCue slides up from below the frame
PunchEach word punches in with emphasis, timed to speech
StampEach word stamps in with a scale pop
TypewriterText types on character by character per cue

Spotlight, Highlighter, Impact, Karaoke, and Punch use word-level timestamps from transcription. They require a generate made after word-timestamp support was added. If an older generate lacks word timestamps, re-generate the captions.

Troubleshooting

Panel doesn't appear in Window → Extensions

Re-run Install-Cuemark.cmd or Install-Cuemark.command from a fresh zip extract, then fully quit and relaunch After Effects. See README.txt for manual steps.

Video generate fails on Mac

Re-download Cuemark from the download page and replace the com.captioncraft.panel folder - ffmpeg is bundled for video transcription. If it still fails, install ffmpeg with Homebrew (brew install ffmpeg) or generate from an audio-only file (MP3/WAV).

Blank white panel

Re-copy the panel from a fresh zip download and fully relaunch AE. If still blank, right-click inside the panel and choose Inspect to see any JavaScript errors.

Generate fails with a network error

Check your internet connection. If you're on a corporate network, the firewall may be blocking outgoing requests to keyweaver-backend.vercel.app. Try on a different network or hotspot.

Transcription is inaccurate

Transcription accuracy depends on audio quality. For best results: use clean audio with minimal background noise, ensure the speaker is close to the microphone, and avoid heavy reverb. Use the Text tab to correct any errors after generation.

Captions are out of sync

Make sure the footage item in your AE project matches the file you intended to transcribe. If you've trimmed or sped up the footage in AE after generating, the timestamps will no longer match. Re-generate from the source file.

After Effects freezes during generate

Building many caption layers can take a few seconds. AE may appear to hang - wait for the status message to finish. If it freezes consistently, try a shorter clip (under 15 minutes) or fewer words per line.

Style changes don't apply

Make sure you have a Cuemark Subtitles layer selected or subtitles already generated. Style updates apply automatically; check the status bar at the bottom of the panel for errors. Text tab edits must be saved before they appear in exports.

FAQ

Does Cuemark work with any language?
Cuemark supports 57 languages. Language is auto-detected - you don't need to specify it. Quality varies by language; English, Spanish, French, German, Japanese, and Chinese have the highest accuracy.
Can I edit the AE layers directly after generating?
Yes - captions are native AE text layers. You can edit them in the Timeline, apply effects, change expressions, or move them like any other layer. The Cuemark panel can still re-apply styles after manual edits, but editing text directly in AE won't sync back to the panel's Text tab.
What happens if I run out of credits mid-generate?
The credit check happens before transcription starts. If you don't have enough credits for the full file, the generate will not proceed and you'll be prompted to top up. You won't be charged partial credits for a failed generate.
Can I use Cuemark on client work?
Yes. Commercial use is permitted on the Credits and Monthly plans. Generated captions are yours - there are no watermarks or attribution requirements.
Does Cuemark work offline?
An internet connection is required to transcribe audio - transcription runs in the cloud. Styling, editing text, and switching presets all work without a connection once captions have been generated.
Will Cuemark keep getting updates?
Yes. Cuemark is actively developed. On Windows, open Keyweaver Manager and click Install on Cuemark to get the latest version. On macOS, download the latest zip from the download page and run the installer again. Fully quit and reopen After Effects after updating.

Using Superconductor

Everything you need to go from a prompt to a video clip on your After Effects timeline.

Installation

Superconductor is a CEP panel for Adobe After Effects. The panel appears under Window → Extensions after you install it and fully restart AE.

Full step-by-step instructions are on the download page.

Windows (recommended)

Download Keyweaver Installer from the download page and run Keyweaver-Setup.exe. Open Keyweaver Manager, click Install on Superconductor, then fully quit and reopen After Effects.

If Windows shows SmartScreen, click More info → Run anyway and confirm the publisher is Keyweaver Ltd.

Windows (zip fallback)

Download the Superconductor zip from the download page, extract it, and double-click Install-Superconductor.cmd. The installer copies files to your user AppData folder only.

macOS

Extract the zip and double-click Install-Superconductor.command. If macOS blocks it, right-click → Open → Open once.

After installing, you must fully quit and relaunch After Effects. Restarting is required - closing and reopening a panel is not enough.

Your first generate

1

Open the panel

In After Effects, go to Window → Extensions → Keyweaver - Superconductor. Sign in with your Keyweaver account, or start with guest trial credits.

2

Open your edit comp

Open the composition you want the clip in. Use Refresh active comp if the panel needs to pick up the current aspect ratio.

3

Write a prompt and set options

On the Generate tab, describe the shot. Choose style, quality (Standard or Pro), aspect ratio, duration (5s or 9s), and output size. The panel shows the credit cost before you run.

4

Generate, then import

Click Generate video. When the job finishes, open the Library tab and click Import to place the clip in your active composition.

Import does not consume credits - only a successful generate does. Failed provider runs are refunded.

Credits & account

Superconductor uses your shared Keyweaver Credits balance - the same pool as Cuemark, Trillian, and Tamborine. Exact cost is shown in the panel before you generate.

Typical base costs (before optional upscale): Standard 5s = 8 credits, Standard 9s = 14, Pro 5s = 20, Pro 9s = 36. Choosing 1080p or 4K output can add upscale credits on Standard jobs - the Cost line always reflects your current settings.

Top up or manage your subscription at keyweaver.io/account. Sign in with the same email in the panel and on the website.

Open Superconductor in After Effects to try guest trial credits without an account. Sign up free for 40 Keyweaver Credits shared across the AI plugins.

Generate tab

This is where you describe the shot and kick off generation.

Prompt and enhance

Write what you want to see. Enhance prompt for better results (on by default) expands short prompts into clearer generation instructions.

Style, quality, aspect, duration

  • Style - optional presets such as Motion plate, Cinematic B-roll, Abstract background, Product backdrop, or Social cutaway
  • Quality - Standard or Pro
  • Aspect - Active comp, 16:9, 9:16, or 1:1
  • Duration - 5 seconds or 9 seconds
  • Output size - Active comp (fit on import), 720p, 1080p, or 4K AI upscale

Image-to-video

Enable Use current frame as start to seed the clip from the frame under the playhead in your active composition.

Do not prompt for copyrighted characters, brand logos, public-figure impersonation, deepfakes, or unsafe content. Blocked prompts will not generate.

Library tab

Completed jobs appear in Your library. Click Refresh if a job just finished. When a clip is ready, click Import to add it to the active composition.

Jobs may show as Failed, Refunded, or Blocked. Refunded jobs return credits to your balance.

Troubleshooting

Panel doesn't appear in Window → Extensions

Re-run the installer from a fresh zip extract (or reinstall from Keyweaver Manager), then fully quit and relaunch After Effects.

Blank white panel

Re-copy the panel from a fresh zip download and fully relaunch AE. If still blank, right-click inside the panel and choose Inspect to see any JavaScript errors.

Active comp not available

Open a composition in After Effects, then click Refresh active comp in the panel.

Generate fails with a network error

Check your internet connection. Corporate firewalls may block keyweaver-backend.vercel.app - try another network if needed.

Not enough credits

The credit check happens before generate. Top up at Account or see Pricing.

FAQ

Can I use generated clips on client work?
Yes. Commercial use is permitted on paid credit plans. Follow the panel safety rules - no copyrighted characters, logos, or public-figure impersonation.
Do I need an account?
You can start with guest trial credits in the panel. Create a free account for 40 shared Keyweaver Credits that follow you across Cuemark, Superconductor, Trillian, and Tamborine.
Does Superconductor work offline?
No. Video generation runs in the cloud and needs an internet connection. Importing a clip you already generated still requires the panel and a local file from your library.
How do I update Superconductor?
On Windows, open Keyweaver Manager and click Install on Superconductor. On macOS, download the latest zip from the download page and run the installer again. Fully quit and reopen After Effects after updating.

Everything you need to go from a script to narration audio inside After Effects.

Your first generate

1

Open the panel

In After Effects, go to Window → Extensions → Keyweaver - Trillian. Sign in, or start with guest trial credits.

2

Write or paste your script

On the Generate tab, enter the narration. The panel shows character count and estimated credits as you type (scripts are capped at 2500 characters).

3

Pick a voice

Browse or search the voice list. Filter by gender or accent, preview a voice, then select it.

4

Generate and import

Confirm the cost, click Generate voiceover, then open the Library tab and click Import to place the audio in your active composition.

If Cuemark is installed, Library also offers Import + captions (Free) to hand the take to Cuemark for subtitles without spending extra Trillian credits.

Credits & account

Trillian uses shared Keyweaver Credits. Cost scales with script length - the panel estimates as you type. Short scripts start at a few credits (minimum applies); longer scripts cost more.

Credits are only consumed when you generate voiceover. Importing from the library does not charge again.

Manage your balance at keyweaver.io/account.

Guest trial credits are available in the panel without an account. Sign up free for 40 Keyweaver Credits shared across Cuemark, Superconductor, Trillian, and Tamborine.

Generate tab

Enter your script, choose a voice, and generate narration.

Script

Paste dialogue or narration into the script field. Keep it under 2500 characters. The meta line shows character count and credit estimate.

Voices

Search by name, filter by gender or accent, and sort by Most popular or A–Z. Preview a voice before you generate.

Generate voiceover

Generate is enabled when you have a non-empty script, a selected voice, and enough credits. The finished take is saved to your library.

Library tab

Find past takes under Your library. Use Refresh after a generate finishes.

  • Import - places the audio in your active composition
  • Import + captions (Free) - imports audio and opens Cuemark to caption it (Cuemark must be installed)

Troubleshooting

Panel doesn't appear in Window → Extensions

Reinstall from Keyweaver Manager or a fresh zip, then fully quit and relaunch After Effects.

Generate button stays disabled

Check that the script is not empty, a voice is selected, the script is under 2500 characters, and the voice provider is ready.

Import + captions says install Cuemark

Install Cuemark from Keyweaver Manager (or the Cuemark zip), then try again. Caption import is free once Cuemark is available.

Import fails

Make sure a composition is open and active in After Effects, then retry Import.

Network or auth errors

Confirm you can reach keyweaver-backend.vercel.app. Sign out and back in if your session expired.

FAQ

How are credits charged?
Trillian shows an estimate before you generate. Cost scales with script length and comes from your shared Keyweaver Credits balance.
Where does the audio go?
Generated narration is saved to your library. Import places it in the active After Effects composition so you can time it with picture.
Can I caption a Trillian take?
Yes. Use Import + captions in the Library tab if Cuemark is installed. That handoff does not charge Trillian credits again.
How do I update Trillian?
On Windows, use Keyweaver Manager. On macOS, download the latest zip from the download page and run the installer again. Fully quit and reopen After Effects after updating.

Using Tamborine

Everything you need to generate music beds and sound effects for your After Effects edit.

Tamborine is a CEP panel for Adobe After Effects. The panel appears under Window → Extensions after you install it and fully restart AE.

Full step-by-step instructions are on the download page.

Windows (recommended)

Windows (zip fallback)

Your first generate

1

Open the panel

In After Effects, go to Window → Extensions → Keyweaver - Tamborine. Sign in, or start with guest trial credits.

2

Choose Music or Sound effects

Use the Music tab for beds and underlays, or Sound effects for one-shots, whooshes, and accents.

3

Describe the sound

Write a prompt, set duration and options, and check the credit cost shown in the panel.

4

Generate and import

Click generate, then open the Library tab and click Import to place the audio in your active composition.

Import does not consume credits - only a successful generate does.

Credits & account

Tamborine uses shared Keyweaver Credits. Cost scales with duration - short beds and SFX start from a few credits; longer music costs more. The Cost line updates as you change settings.

Manage your balance at keyweaver.io/account.

Guest trial credits are available in the panel without an account. Sign up free for 40 Keyweaver Credits shared across Cuemark, Superconductor, Trillian, and Tamborine.

Music tab

Generate music beds for underlays and scene atmosphere.

Prompt and option chips

Describe the track you want. Optional chips for Genre, Mood, Video Theme, and Instrument help shape the AI prompt without cluttering the text box.

Duration and instrumental

Set length from a few seconds up to two minutes. Enable Instrumental only when you need beds without vocals.

Generate music

Confirm the cost, then click Generate music. The finished bed appears in your library.

Sound effects tab

Generate one-shot effects for hits, whooshes, transitions, and accents.

Description and length

Describe the effect. Choose Auto length (provider decides) or a Custom length.

Loop and influence

Seamless loop is useful for beds or repeating accents. Prompt influence controls how strongly the description drives the result.

Generate sound effect

Confirm the cost, then click Generate sound effect.

Library tab

Browse All, Music, or SFX. Click Refresh after a generate finishes, then Import to place audio in the active composition.

Troubleshooting

Network or auth errors

Using Ludo

Everything you need to animate, arrange, and export everyday After Effects work from one panel.

Installation

Ludo is a CEP panel for Adobe After Effects. The panel appears under Window → Extensions after you install it and fully restart AE.

Full step-by-step instructions are on the download page.

Windows (recommended)

Download Keyweaver Installer from the download page and run Keyweaver-Setup.exe. Open Keyweaver Manager, click Install on Ludo, then fully quit and reopen After Effects.

Windows (zip fallback)

Download the Ludo zip, extract it, and double-click Install-Ludo.cmd.

macOS

Extract the zip and double-click Install-Ludo.command. If macOS blocks it, right-click → Open → Open once.

After installing, you must fully quit and relaunch After Effects. Restarting is required - closing and reopening a panel is not enough.

First steps

1

Open the panel

In After Effects, go to Window → Extensions → Keyweaver - Ludo.

2

Select a layer or keys

Open a composition and select the layers or keyframes you want to work on. Ludo enables tools based on what you have selected.

3

Pick a tool category

Expand Animation, Layer, Text, Colour, Layout, Extras, or Export and apply a tool. Changes write to native AE properties and layers.

Ludo does not use Keyweaver Credits. You can use the full toolkit without signing in for credits.

Credits & account

Ludo does not draw from your Keyweaver Credits balance. Animation, keyframe, text, layout, colour, and export tools run locally in After Effects.

You can still sign in from the account menu to manage your Keyweaver account, or open the other AI plugins from Extras.

Animation & keys

Use Animation controls for ease curves and live expressions, plus keyframe helpers.

Easing

Choose an ease curve (Linear, Bezier, Smooth, Soft, Natural, Punchy, Snappy, Whip, Bounce, Overshoot, Spring, and more). Target Start, End, or All keys, and apply In, Out, or All directions.

Keyframe tools

With multiple keys selected across layers: Stagger, Align, Reverse, Distribute, and Mirror. Stagger includes a gap control.

Layer controls

Apply transitions, loops, and simple physics-style motion to selected layers.

  • Transitions - Fade, Slide, Scale, Drop, Bounce, Roll, Blur, and more (in or out)
  • Looping - Float, Pulse, Shake, Drift, Spin, Wiggle
  • Physics / Collider - optional motion helpers with clear/reset controls

Text tools

Select a text layer to unlock text helpers.

  • Split characters - break glyphs into layers (optional precomp)
  • Stagger / Animate on - Fade on, Slide up, Typewriter, Cascade, Smooth in, and more
  • Blink cursor - available when Typewriter is on
  • Text box - stroke box with padding, width, roundness, and colour
  • Copy style to all text - match the selected text layer’s style across other text in the comp

Layout & colour

Layout

Use the Cloner (Linear, Grid, or Radial) with live effectors for scale, spacing, opacity, rotation, and time. Arrange selected layers in a grid, or open Effect Controls for the cloner driver.

Colour palette

Save swatches, sample from the comp, and apply colours to text, solids, shapes, lights, and Color Controls. Swatches are stored locally in the panel.

Extras

Launch Cuemark, Superconductor, Trillian, or Tamborine from Ludo - or open Keyweaver Manager if a plugin is missing.

Export

Still frame

Export the current frame as PNG, JPEG, TIFF, or PSD with a save dialog. The Render Queue window stays out of the way.

GIF

Export the work area as a GIF. Choose scale, frame skip, and compression - the panel shows a live size estimate. GIF export needs ffmpeg (bundled with the Ludo install, or it can fall back to Cuemark’s bundled copy if present).

Troubleshooting

Panel doesn't appear in Window → Extensions

Reinstall from Keyweaver Manager or a fresh zip, then fully quit and relaunch After Effects.

Tools are greyed out

Open a composition and select a layer or keyframes that match the tool. Text tools need a text layer; keyframe tools need selected keys.

GIF export fails

Confirm ffmpeg is available (reinstall Ludo, or install Cuemark which also bundles ffmpeg), then try again.

Blank white panel

Re-copy the panel from a fresh zip and fully relaunch AE. If still blank, right-click inside the panel and choose Inspect.

FAQ

Does Ludo use credits?
No. Ludo does not draw from your Keyweaver Credits balance.
What is included?
Animation easing, keyframe tools, layer transitions and loops, text helpers, layout/cloner, colour palette, frame export, and work-area GIF export.
Are the results editable in After Effects?
Yes. Ludo writes native properties, expressions, and layers - you can keep editing everything in the timeline after you apply a tool.
How do I update Ludo?
On Windows, use Keyweaver Manager. On macOS, download the latest zip from the download page and run the installer again. Fully quit and reopen After Effects after updating.
